## Usage

Format files:
`yarn format`

Lint files:
`yarn lint`

Compile files:
`yarn compile`

Run tests:
`yarn test`

Verify contract to bsc testnet:
`yarn deploy-testnet`  - put your mnemonic to `.secrets.json`

Verify contract on bscscan:
`yarn verify-testnet` - put your bscscan api key to `.secrets.json`
